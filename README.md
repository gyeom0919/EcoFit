## 프로젝트 개요 
일회용 플라스틱이라는 일상생활속의 환경문제를 개선하기위해 환경켐페인 연계 시스템 EcoFit




## 예상 기능 (구현한 것 ⭕️ 표시)
```
Map Kit
 ㄴ 애플 맵 구현 ⭕️
 ㄴ 현재 주소 뜨기 ⭕️
 ㄴ 매장별, 지역별 이벤트 정리
 
Quest View
 ㄴ TableView 퀘스트리스트 ⭕️
 ㄴ 퀘스트에 각각 이미지 ⭕️
 ㄴ 조건과 마일리지 ⭕️
 
Item View 
 ㄴ CollectionView 아이템 구성 ⭕️
 ㄴ 아이템 ⭕️
 ㄴ 구매기능
 
Profile View
 ㄴ 보유 마일리지 ⭕️
 ㄴ 환경 친화력 ⭕️
 ㄴ 세부 화면들
 
APP 
 ㄴ DB 연결
 ㄴ 백엔드 연결
```
